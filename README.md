# Minim1 - RPN Calculator

En este proyecto he hecho un sistema para gestionar operaciones matemáticas en notación polaca inversa (RPN).

La aplicación permite:
- añadir institutos
- añadir alumnos
- pedir operaciones matemáticas
- procesarlas en orden de llegada
- consultar las operaciones de un alumno
- consultar las operaciones de un instituto
- ver un ranking de institutos según el número de operaciones realizadas

## Tecnologías usadas

- Java 17
- Maven
- JUnit 5
- Log4j2
- Spring Boot
- Swagger
